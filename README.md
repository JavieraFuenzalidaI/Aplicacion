Integrantes: Javiera Fuenzalida y Catalina Villegas.
Descripción: Mascota que invita a completar metas para mejorar la calidad de vida,las tareas pueden ser relacionadas a diferentes ámbitos: laboral, académica, relacional, etc.
La mascota cambia de animo a medida que las tareas se van realizando.
Funcionalidades implementadas: 
- Sistema de usuarios (registro, login, sesión persistente)
- Base de datos SQLite para almacenar usuarios
- Roles de usuario (admin y usuario normal)
- Listado de usuarios (solo admin)
- Pantalla de mascotas
- Persistencia adicional con SharedPreferences
- Navegación y estructura completa con Compose
Pasos para ejecutar: Importe el proyecto en Android Studio, una vez todo este instalado ejecute la aplicación en el dispositivo virtual o dispositivo fisico.
