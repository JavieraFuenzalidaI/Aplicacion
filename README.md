## Funcionalidades

La aplicación permite la gestión de usuarios, incluyendo:

*   Inicio de sesión de usuarios.
*   Registro de nuevos usuarios.
*   Visualización de la lista de usuarios.
*   Eliminación de usuarios.
*   Actualización de la información de los usuarios con diferentes niveles de permisos (administrador y moderador).

## Endpoints Utilizados

La aplicación se comunica con un backend propio a través de los siguientes endpoints:

*   **URL Base:** `http://3.84.192.34:3000/`

*   **Endpoints específicos:**
    *   `POST /login`
    *   `POST /usuarios`
    *   `GET /usuarios`
    *   `GET /usuarios/{id}`
    *   `DELETE /usuarios/{id}`
    *   `PATCH /usuarios/admin/{id}`
    *   `PATCH /usuarios/moderador/{id}`

## Instrucciones para Ejecutar el Proyecto

1.  Clona el repositorio.
2.  Abre el proyecto con Android Studio.
3.  Asegúrate de tener un emulador de Android configurado o un dispositivo físico conectado.
4.  Ejecuta la aplicación desde Android Studio.
